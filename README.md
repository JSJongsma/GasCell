# GasCell
This is a model for gas absorption spectroscopy. The model tries to find mirror and hole configurations (i.e. positions, angles etc.) for which a certain amount of reflection/pathlength/intensity is found. 
The model has few dependencies (on NumPy, Matplotlib, etc.). The number of mirrors, holes (in the mirrors), and their dimensions are completely arbitrary, although the writer works with 3 mirrors with 2 holes (exit/entry).
